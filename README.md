# Kvartinder

### Kvartinder is a graduate project from ADS of Yandex 2018. This app utilizes Yandex API for YandexRealty (search for realty in Russia) using new design similar to Tinder. 

### Details:
  - Kotlin
  - Clean Architecture
  - MVVM
  - Android Architecture Components

### Libraries used:
  - RxJava2
  - Dagger2
  - Room
  - Kotlin Coroutines
  - Retrofit
  - DelegateAdapter
  - Hamcrest
  - Mockito
  - Tinder Cards
